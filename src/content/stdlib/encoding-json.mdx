<Callout type="analogy" title="📦 Analogy">
JSON is a shipping container for data: `Marshal` packs a Go value into the standard box so it can travel over a network or into a file, and `Unmarshal` unpacks a box back into a Go value. Struct tags are the packing slip — they say which Go field maps to which label on the box.
</Callout>

## Marshal and Unmarshal

`Marshal` turns a Go value into `[]byte` of JSON; `Unmarshal` fills a Go value from JSON bytes. **Struct tags** map fields to JSON keys — conventionally written with backticks:

```go
type User struct {
	Name  string `json:"name"`
	Email string `json:"email,omitempty"`
	Age   int    `json:"age"`
	pass  string // unexported: never marshaled
}
```

Run a round-trip (the playground writes the tags as plain string literals, which work identically):

<Playground client:visible title="json.go" code={String.raw`package main

import (
	"encoding/json"
	"fmt"
)

type User struct {
	Name  string "json:\"name\""
	Email string "json:\"email,omitempty\""
	Age   int    "json:\"age\""
	pass  string // unexported: never marshaled
}

func main() {
	u := User{Name: "Ada", Age: 36, pass: "secret"}
	b, _ := json.Marshal(u)
	fmt.Println(string(b)) // {"name":"Ada","age":36} — email omitted, pass skipped

	pretty, _ := json.MarshalIndent(u, "", "  ")
	fmt.Println(string(pretty))

	// decode JSON back into a struct (Unmarshal needs a pointer)
	src := "{\"name\":\"Bob\",\"email\":\"b@x.io\",\"age\":40}"
	var got User
	if err := json.Unmarshal([]byte(src), &got); err != nil {
		fmt.Println("error:", err)
	}
	fmt.Printf("%+v\n", got)
}`} />

## Known shape? Use a struct. Unknown? Use a map

<Mermaid code={String.raw`graph LR
  J["JSON bytes"] -->|"Unmarshal into &struct"| S["typed struct (you know the shape)"]
  J -->|"Unmarshal into &map[string]any"| M["generic tree: object→map, array→[]any, number→float64"]`} />

For dynamic JSON, decode into `map[string]any` — but reach for a typed struct whenever you know the shape; it's safer and self-documenting.

## Streaming with Encoder/Decoder

For an `io.Reader`/`io.Writer` (a file, an HTTP body) use `json.NewDecoder(r)` / `json.NewEncoder(w)` instead of reading everything into memory first:

```go
// decode an HTTP request body straight into a struct
var u User
err := json.NewDecoder(r.Body).Decode(&u)

// encode a value straight to the response
json.NewEncoder(w).Encode(u)
```

<Callout type="gotcha" title="⚠️ Unmarshal needs a pointer, and ignores what it can't match">
Pass a **pointer** to `Unmarshal` (`&u`) — it has to write into your value. Decoding is lenient: JSON keys with no matching field are silently dropped, and missing fields keep their zero value, so always check the returned `error` and validate. Numbers default to `float64` inside an `any`, which surprises people expecting `int`. For custom formats, implement `json.Marshaler` / `json.Unmarshaler`.
</Callout>
